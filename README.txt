README - Proyecto MATLAB: Ejercicios de Simulación y Cálculo Numérico
=================================================================

Este proyecto contiene los ejercicios prácticos solicitados en el examen de MATLAB.
Incluye las funciones originales SIN MODIFICAR y un script general para ejecutarlas.

Contenido del paquete:
----------------------
1. paseo2D.m          → Función que simula un paseo aleatorio continuo en 2D.
2. paseo2D_malla.m    → Paseo aleatorio discreto sobre una malla de coordenadas enteras.
3. collatz.m          → Calcula la longitud de la secuencia de Collatz para un número entero.
4. soluciones_examen.m → Script principal que ejecuta todos los ejercicios del examen.
5. README.txt          → Este archivo con la descripción del proyecto.

Instrucciones de uso:
---------------------
1. Abrir MATLAB y establecer la carpeta de trabajo donde se haya descomprimido este paquete.
2. Ejecutar el archivo "soluciones_examen.m" para correr todos los ejercicios.
3. Las figuras y resultados se mostrarán automáticamente.
4. NO modificar el código original de las funciones, según las indicaciones del examen.
